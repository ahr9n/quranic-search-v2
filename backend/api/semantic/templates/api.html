This homepage of Quranic Semantic Search API DOCS.

http://localhost:5000/api/semantic/similar-word/{word}

Take a string of a single word, to return the first 100 most similar words, containing the meaning of the given word.

http://localhost:5000/api/semantic/similar-verse/{query}

Take a query (string of one or more words), to return the first 50 most similar verses from the Holy Quran, containing the meaning of the given query.

Note: You must run the lexical search API first, as the results of verses properties are retrieved from the lexical search API by the verses IDs from the semantic search API.
